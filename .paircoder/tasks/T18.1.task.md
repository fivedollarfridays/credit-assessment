# Objective

Replace in-memory `_users` dict and `_reset_tokens` OrderedDict with UserRepository and ResetTokenRepository. Users and credentials survive restarts.

# Implementation Plan

1. Update user_store.py functions to use repository (async session from app state)
2. Add reset token expiry (TTL — 30 minutes default, configurable)
3. Store reset token created_at timestamp; check TTL on pop
4. Update all callers (user_routes.py, admin_routes.py, dashboard.py, data_rights_routes.py, roles.py)
5. Handle async transition — functions become async where needed
6. Keep in-memory fallback for tests (or use test DB)
7. Update all test mocks/patches

# Acceptance Criteria

- [x] _users dict removed; all user CRUD goes through UserRepository
- [x] _reset_tokens removed; reset tokens stored in DB with created_at
