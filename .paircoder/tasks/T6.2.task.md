# Objective

Create privacy policy and terms of service documents, with API endpoints to serve them.

# Implementation Plan

1. Draft privacy policy covering data collection, use, retention, sharing
2. Draft terms of service covering API usage, SLA, liability
3. Create `/legal/privacy` and `/legal/terms` endpoints
4. Add acceptance tracking (users must accept ToS)
5. Version the legal documents for change tracking

# Acceptance Criteria

- [ ] Privacy policy covers data handling practices
- [ ] Terms of service covers API usage terms
- [ ] Legal documents accessible via API endpoints
- [ ] ToS acceptance tracked per user
- [ ] Documents versioned for change tracking
