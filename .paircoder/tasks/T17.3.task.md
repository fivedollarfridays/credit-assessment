# Objective

Create async repository classes for every model that will replace an in-memory store.

# Implementation Plan

Create repository classes (in repository.py or split into multiple files if needed):

1. **UserRepository** — create, get_by_email, get_all, count, update, set_role, set_password_hash, delete
2. **ResetTokenRepository** — store, pop (with expiry check), prune_expired
3. **SubscriptionRepository** — create/update, get_by_email, list, count_active
4. **ConsentRepository** — record, check, withdraw, get_by_user
5. **UserAssessmentRepository** — record, get_by_user, delete_by_user, purge_expired
6. **WebhookRepository** — create, get, list, list_by_owner, get_subscribed, delete, count
7. **WebhookDeliveryRepository** — log_delivery, get_by_webhook, get_by_status
8. **ApiKeyRepository** — create, lookup (with expiry check), revoke, list_by_org, prune_expired
9. **FeatureFlagRepository** — create, get, list, update, delete
10. **AuditRepository** — expand existing with query methods (by action, by org, date range, purge)
