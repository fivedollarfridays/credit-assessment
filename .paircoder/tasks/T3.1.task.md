# Objective

Enforce HTTPS/TLS in production via a reverse proxy (Caddy) in docker-compose and HSTS middleware in FastAPI.

# Implementation Plan

1. Add Caddy reverse proxy service to docker-compose with automatic TLS
2. Add HTTPS redirect middleware to FastAPI for production mode
3. Add HSTS headers to all responses in production
4. Create `docker-compose.prod.yml` override for production TLS config
5. Test with self-signed certs locally

# Acceptance Criteria

- [ ] All HTTP requests redirect to HTTPS in production
- [ ] HSTS header present on all production responses
- [ ] Caddy handles TLS termination in docker-compose
- [ ] Development mode still works on HTTP
- [ ] Tests pass locally (`pytest`)
