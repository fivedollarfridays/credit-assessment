# Objective

Replace in-memory _org_assessments with DB queries on AssessmentRecord (now has org_id + user_id). Add GET /v1/assessments history endpoint.

# Implementation Plan

1. Update _persist_assessment() to save org_id (from auth identity)
2. Replace store_org_assessment() with DB insert (already done via _persist_assessment)
3. Replace get_org_assessments() with DB query filtered by org_id
4. Replace count_all_assessments/count_org_assessments with SQL COUNT
5. Replace delete_user_org_assessments with DB DELETE
6. Remove _org_assessments dict
7. Add GET /v1/assessments endpoint with pagination (limit/offset)
   - Filter by org_id (from auth), optional date range
   - Requires auth
8. Update dashboard.py, data_rights.py callers
9. Update tests
