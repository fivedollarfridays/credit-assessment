# Objective

Generate and publish client SDKs for Python, JavaScript, and Go from the OpenAPI specification.

# Implementation Plan

1. Use openapi-generator or similar to auto-generate client libraries
2. Add type hints, docstrings, and error handling to generated code
3. Create SDK-specific README with installation and usage examples
4. Publish to PyPI (Python), npm (JavaScript), and pkg.go.dev (Go)
5. Set up CI to regenerate SDKs when OpenAPI spec changes

# Acceptance Criteria

- [x] Python SDK installable via pip with typed client
- [x] JavaScript SDK installable via npm with TypeScript types
- [x] Go SDK available with idiomatic error handling
- [x] All SDKs include authentication helpers
- [x] SDK generation automated in CI
