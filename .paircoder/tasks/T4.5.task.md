# Objective

Ensure all data is scoped per organization. API keys map to orgs, and queries are automatically filtered by org_id.

# Implementation Plan

1. Add Organization model with org_id, name, created_at
2. Add org_id foreign key to all tenant-scoped tables
3. Create middleware that resolves org_id from JWT/API key and injects into request state
4. Update repository layer to always filter by org_id
5. Add admin-only cross-org query capability
6. Write tests verifying cross-tenant data isolation

# Acceptance Criteria

- [x] All assessment data scoped to org_id
- [x] Org A cannot access Org B's data
- [x] API keys automatically resolve to correct org
- [x] Admin role can query across orgs
- [x] Database queries always include org_id filter
- [x] Tests verify cross-tenant isolation
- [x] Tests pass locally (`pytest`)
