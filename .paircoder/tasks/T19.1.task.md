# Objective

Replace in-memory _audit_entries deque with AuditRepository. Audit trail survives restarts and can meet FCRA 7-year retention.

# Implementation Plan

1. Expand AuditLog ORM model to match current audit entry fields (user_id_hash, request_summary, result_summary, org_id)
2. Update create_audit_entry() to write via AuditRepository
3. Update get_audit_trail() to query via AuditRepository (with filtering)
4. Update purge_audit_trail() to use DB DELETE with date filter
5. Update count_audit_entries() to use SQL COUNT
6. Remove _audit_entries deque
7. Update admin_routes.py and dashboard.py callers
8. Update tests

# Acceptance Criteria

- [ ] _audit_entries deque removed
