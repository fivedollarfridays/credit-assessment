## Objective

Add `slowapi` to the runtime dependencies in `pyproject.toml`. It's imported by `router.py` but not declared, so any fresh `pip install` will fail.

In `pyproject.toml` under `[project] dependencies`, add:
```
"slowapi>=0.1.9",
```
