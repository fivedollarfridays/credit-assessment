# Objective

Implement GDPR and CCPA compliance: consent management, data export, data deletion, and right-to-be-forgotten endpoints.

# Implementation Plan

1. Add consent tracking to user model (consent_given_at, consent_version)
2. Create data export endpoint: GET /v1/user/data-export (returns all user data as JSON)
3. Create data deletion endpoint: DELETE /v1/user/data (right to be forgotten)
4. Implement data retention policies with automatic purging
5. Add cookie consent tracking for any future web UI
6. Write tests for export, deletion, and retention

# Acceptance Criteria

- [x] Users can export all their data via API
- [x] Users can request data deletion
- [x] Deletion cascades to all related records
- [x] Data retention policy automatically purges old data
- [x] Consent tracked with version and timestamp
- [x] Tests pass locally (`pytest`)
