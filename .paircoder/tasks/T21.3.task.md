# Objective

Expose the score simulation engine via `POST /v1/simulate`. Accepts a CreditProfile + list of proposed actions, returns projected score, readiness, and eligibility changes.

# Implementation Plan

1. Create `SimulationRequest` Pydantic model in a new `simulate_routes.py`:
   - profile: CreditProfile (required)
   - actions: list[SimulationAction] (required, min 1, max 10)
2. Create `POST /simulate` endpoint:
   - Auth required (Depends(verify_auth))
   - Rate limited (30/minute, same as assess)
   - Calls ScoreSimulator.simulate()
   - Returns SimulationResult
3. Also create `POST /simulate/simple` variant:
   - Accepts SimpleCreditProfile instead of full CreditProfile
   - Converts via to_credit_profile() then simulates
4. Register router in router.py under `/v1/simulate`
5. Add FCRA disclaimer to SimulationResult (projections are estimates, not guarantees)
