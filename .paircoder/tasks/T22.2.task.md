# Objective

Create a letter generation engine that takes a NegativeItem + template type + bureau and produces a complete, ready-to-send dispute letter with legal citations, creditor addressing, and structured output.

# Implementation Plan

1. Create `letter_generator.py` with `LetterGenerator` class
2. Define `LetterRequest` Pydantic model:
   - negative_item: NegativeItem (structured item to dispute)
   - letter_type: LetterType
   - bureau: Bureau
   - consumer_name: str (required)
   - consumer_address: str | None (optional)
   - account_number: str | None (optional, for creditor reference)
   - variation: int | None (optional, random if not specified)
3. Define `GeneratedLetter` Pydantic model:
   - subject: str
   - body: str (full letter text)
   - bureau: Bureau
   - bureau_address: str
   - legal_citations: list[str]
   - letter_type: LetterType
   - negative_item_description: str
   - generated_at: str (ISO timestamp)
   - disclaimer: str
4. Implement `LetterGenerator.generate(request: LetterRequest) -> GeneratedLetter`:
   - Select template (by type + variation)
   - Fill placeholders from request + negative item fields
   - Format bureau address
   - Attach legal citations
   - Add FCRA/CROA disclaimer
5. Implement `LetterGenerator.generate_batch(requests: list[LetterRequest]) -> list[GeneratedLetter]`:
   - Generate multiple letters, auto-vary templates to avoid detection
