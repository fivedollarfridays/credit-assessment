# Objective

Fix XSS vulnerability in `src/modules/credit/agents/export.py` where all 7 `_render_*` functions interpolate user data into HTML without `html.escape()`.

# Implementation Plan

1. Add `import html` and `_esc = html.escape` helper to export.py
2. Wrap every user-controlled value in `_esc()` across all render functions + footer
3. Add XSS-specific tests to test_export.py
